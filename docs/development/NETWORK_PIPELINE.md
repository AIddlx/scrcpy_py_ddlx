## 4. 数据包格式

### 4.1 UDP 数据包结构

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        UDP 数据包 (最大 65507 字节)                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  ┌─────────────────────────────────────┐                               │
│  │          UDP Header (16字节)         │                               │
│  ├─────────────────────────────────────┤                               │
│  │ Offset  Size  Field                  │                               │
│  │ 0       4     sequence_number        │  包序号 (递增)                │
│  │ 4       8     timestamp              │  时间戳 (纳秒)                │
│  │ 12      4     flags                  │  标志位                       │
│  └─────────────────────────────────────┘                               │
│                                                                         │
│  ┌─────────────────────────────────────┐                               │
│  │       Scrcpy Header (12字节)         │                               │
│  ├─────────────────────────────────────┤                               │
│  │ Offset  Size  Field                  │                               │
│  │ 0       8     pts_flags              │  PTS + 标志位                 │
│  │               bit63: CONFIG标志      │  1=配置包, 0=媒体帧           │
│  │ 8       4     payload_size           │  负载大小                     │
│  └─────────────────────────────────────┘                               │
│                                                                         │
│  ┌─────────────────────────────────────┐                               │
│  │          Payload (N字节)             │                               │
│  ├─────────────────────────────────────┤                               │
│  │ 配置包: codec_id(4) + width(4) + height(4)                          │
│  │ 或     编码器extradata (SPS/PPS等)                                  │
│  │ 媒体帧: H.264/H.265 NAL单元                                         │
│  └─────────────────────────────────────┘                               │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 4.2 Flags 字段定义

```
UDP Flags (4字节, big-endian):
┌────────────────────────────────────────────────────────────────┐
│ bit 31    │ bit 1    │ bit 0                                    │
│ FRAGMENTED│ KEY_FRAME│ CONFIG                                    │
└────────────────────────────────────────────────────────────────┘

- CONFIG (bit 0): 1=配置包, 0=媒体帧
- KEY_FRAME (bit 1): 1=关键帧, 0=P/B帧
- FRAGMENTED (bit 31): 1=分片包, 0=完整包
```

## 5. 服务端数据流

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                                  服务端数据流                                     │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│  ┌─────────────┐      ┌─────────────┐      ┌─────────────┐      ┌───────────┐  │
│  │   Screen    │      │   MediaCodec │      │   Streamer  │      │UdpSender  │  │
│  │  Capture    │─────►│   Encoder    │─────►│  (包装器)   │─────►│  (UDP)    │  │
│  └─────────────┘      └─────────────┘      └─────────────┘      └───────────┘  │
│        │                    │                    │                    │        │
│        │                    │                    │                    │        │
│        ▼                    ▼                    ▼                    ▼        │
│  ┌─────────────┐      ┌─────────────┐      ┌─────────────┐      ┌───────────┐  │
│  │VirtualDisplay│     │ H.264/H.265 │      │ 添加Scrcpy  │      │ 添加UDP   │  │
│  │ (Surface)   │      │ NAL Units   │      │ 头部        │      │ 头部      │  │
│  └─────────────┘      └─────────────┘      └─────────────┘      └───────────┘  │
│                                                                                 │
│  ┌─────────────┐      ┌─────────────┐      ┌─────────────┐      ┌───────────┐  │
│  │   Audio     │      │   Opus/AAC  │      │   Streamer  │      │UdpSender  │  │
│  │  Capture    │─────►│   Encoder   │─────►│  (包装器)   │─────►│  (UDP)    │  │
│  └─────────────┘      └─────────────┘      └─────────────┘      └───────────┘  │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

详细流程:

1. ScreenCapture
   - 使用 SurfaceControl API 捕获屏幕
   - 输出到 VirtualDisplay 的 Surface

2. MediaCodec Encoder (SurfaceEncoder)
   - 配置: codec, bitrate, fps, i-frame-interval, CBR/VBR
   - 从 Surface 获取编码后的数据
   - 输出: H.264/H.265 NAL Units

3. Streamer
   - 添加 Scrcpy 头部 (pts_flags, payload_size)
   - 处理配置包 (SPS/PPS) 和媒体帧
   - 输出: 完整的 Scrcpy 包

4. UdpMediaSender
   - 添加 UDP 头部 (seq, timestamp, flags)
   - 处理分片 (如果超过 UDP MTU)
   - 发送到客户端 IP:Port
```

## 6. 客户端数据流

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                                  客户端数据流                                     │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                 │
│  ┌───────────┐      ┌─────────────┐      ┌─────────────┐      ┌─────────────┐  │
│  │UDP Socket │      │UdpPacket    │      │UdpVideo     │      │ VideoDecoder│  │
│  │ (接收)    │─────►│Reader       │─────►│Demuxer      │─────►│ (PyAV)      │  │
│  └───────────┘      └─────────────┘      └─────────────┘      └─────────────┘  │
│        │                   │                    │                    │         │
│        ▼                   ▼                    ▼                    ▼         │
│  ┌───────────┐      ┌─────────────┐      ┌─────────────┐      ┌─────────────┐  │
│  │ recvfrom()│      │ 解析UDP头部 │      │ 解析Scrcpy  │      │ H.265解码   │  │
│  │ 64KB缓冲  │      │ seq/ts/flags│      │ 头部        │      │ YUV420P    │  │
│  └───────────┘      └─────────────┘      └─────────────┘      └─────────────┘  │
│                                                                     │         │
│                                                                     ▼         │
│                                                              ┌─────────────┐  │
│                                                              │ OpenGL渲染  │  │
│                                                              │ (QWidget)   │  │
│                                                              └─────────────┘  │
│                                                                                 │
│  ┌───────────┐      ┌─────────────┐      ┌─────────────┐      ┌─────────────┐  │
│  │UDP Socket │      │UdpPacket    │      │UdpAudio     │      │AudioDecoder │  │
│  │ (接收)    │─────►│Reader       │─────►│Demuxer      │─────►│ (PyAV)      │  │
│  └───────────┘      └─────────────┘      └─────────────┘      └─────────────┘  │
│                                                                     │         │
│                                                                     ▼         │
│                                                              ┌─────────────┐  │
│                                                              │SoundDevice  │  │
│                                                              │ (播放)      │  │
│                                                              └─────────────┘  │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘

详细流程:

1. UDP Socket
   - 绑定到本地端口 (27185 视频, 27186 音频)
   - 接收缓冲区: 4MB (视频), 256KB (音频)

2. UdpPacketReader
   - 解析 UDP 头部 (16字节)
   - 提取: sequence, timestamp, flags
   - 处理乱序/丢包

3. UdpVideoDemuxer / UdpAudioDemuxer
   - 解析 Scrcpy 头部 (12字节)
   - 区分配置包和媒体帧
   - 处理分片重组
   - 缓冲管理

4. VideoDecoder (PyAV)
   - 配置解码器 (H.264/H.265)
   - 设置 extradata (SPS/PPS)
   - 解码为 YUV420P 帧

5. AudioDecoder (PyAV)
   - 配置解码器 (Opus/AAC/FLAC)
   - 解码为 PCM float32

6. 输出
   - 视频: OpenGL 渲染到 QWidget
   - 音频: SoundDevice 播放
```

## 10. 与ADB隧道模式对比

| 特性 | ADB隧道模式 | 网络UDP模式 |
|-----|------------|------------|
| 连接方式 | USB + ADB forward | WiFi TCP + UDP |
| 延迟 | 20-50ms | 20-100ms |
| 带宽 | USB带宽 | WiFi带宽 |
| 稳定性 | 高 | 中 (受干扰) |
| 可断开USB | 否 | 是 |
| 热连接支持 | 否 | 是 (规划中) |
| 能力协商 | 否 | 是 |
