# GUI 应用入口 (scrcpy_mcp_gui.py)

> PyQt6 桌面应用，集成 MCP 服务管理

---

## 文件位置

```
scrcpy_mcp_gui.py
```

---

## 运行方式

```bash
python scrcpy_mcp_gui.py
```

或

```bash
python -m scrcpy_py_ddlx.gui
```

---

## 功能模块

### 主窗口

- 设备连接管理
- 连接面板 (USB/网络)
- 设备信息面板
- 媒体控制面板
- 日志面板

### MCP 服务管理

- 启动/停止 HTTP MCP 服务
- 服务状态监控
- 端口配置

### 预览窗口

- 独立进程运行
- 多设备支持
- 实时画面显示

### 文件保存路径 (v1.5 规范)

- 截图: `~/Documents/scrcpy-py-ddlx/screenshots/`
- 录音/视频: `~/Documents/scrcpy-py-ddlx/recordings/`
- 下载文件: `~/Documents/scrcpy-py-ddlx/files/<原路径>`

## 依赖

- PyQt6 或 PySide6
- qt-material (主题)

```bash
pip install PyQt6 qt-material
```

---

## 配置

配置文件位置: `~/.config/scrcpy-py-ddlx/gui_config.json`

```json
{
  "theme": "dark_teal",
  "auto_connect": false,
  "mcp_port": 3359,
  "last_device": "192.168.1.100:5555"
}
```
